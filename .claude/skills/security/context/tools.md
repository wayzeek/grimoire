# Security Tools

## Static Analysis
- **Slither**: `slither .`
- **Aderyn**: `aderyn .`
- **Mythril**: `myth analyze contract.sol`

## Testing
- **Echidna**: Fuzzing tool
- **Foundry**: Built-in fuzzing and invariant testing
- **Certora**: Formal verification

## Runtime Analysis
- **Tenderly**: Transaction simulation and debugging
- **Forta**: Real-time threat detection

## Best Practices
1. Run multiple tools (no single tool catches everything)
2. Manual review is essential
3. Focus on high-severity findings first
4. Understand false positives
5. Keep tools updated
